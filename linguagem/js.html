JavaScript

O que é?

É uma linguagem de programação, criado nos anos 90 pela Netscape Communications, seu objetivo é ajudar na criação de sites mais interativos e dinâmicos, se provavelmente vê sites onde mapas interativos ou gráficos. você pode apostar que o Javascript. podemos marcar um simples bloco de texto usando HTML para dar estrutura. Adicionar um pouco de CSS para deixar nosso parágrafo um pouco mais estruturado. E finalmente, podemos adicionar Javascript para ficar dinâmico.

Javascript é uma linguagem de programação que a princípio, para ser utilizada em navegadores e manipular comportamentos de páginas web. Hoje, ela é uma das mais populares do mundo, o que a faz ser considerada fundamental para programadores e para os seus usuários.

O real nome do Javascript é ECMAScript – porém o nome Javascript é mais comumente usado na prática.

Com JavaScript

O JavaScript permite controlar os elementos de uma página em tempo real, sem necessariamente ter que receber os dados ou uma resposta do servidor. Por exemplo, é possível atualizar o conteúdo de uma página web sem precisar recarregá-la por completo ao preencher um formulário.

O que faz / para que serve?

No começo, páginas da web costumavam ser estáticas, ou seja, exibiam apenas as informações essenciais em um layout fixo, completamente diferente dos sites que vemos hoje em dia. O JavaScript tornou os sites mais dinâmicos, de forma que o usuário possa fazer diversas interações e alterar o layout do conteúdo na página.

Foram estabelecidas bibliotecas, frameworks e formas de programação conforme a linguagem foi recebendo novas versões. Atualmente, podemos usar JavaScript para front-end e back-end, e não apenas em navegadores da web.

Por apresentar todo o conteúdo de uma página de forma interativa e dinâmica, tornando a interação com o usuário mais eficiente, é uma das tecnologias mais importantes para o front-end.

  • Strings

    Strings são usadas para representar vários caracteres, como texto. Elas oferecem vários métodos e funcionalidades para manipular e processar o texto.

    1. Declarando Strings:

    Você pode declarar strings usando aspas simples, duplas ou crases (template literals) em JavaScript. Exemplo:

    2. Concatenar Strings:

    Você pode combinar strings usando o operador (+) ou template literals.

    Você pode obter o tamanho de uma string usando a propriedade length.

    4. Acessando Caracteres:

    Você pode acessar caracteres individuais em uma string usando colchetes e índices, onde o primeiro caractere sempre vai começar com o índice 0.

  • Métodos

    JavaScript fornece diversos métodos embutidos para manipular strings.

    • toUpperCase() e toLowerCase():

      • Convertem uma string para maiúsculas ou minúsculas.

    • split():

      • Divide uma string em um array de substrings com base em um delimitador.

    • substring(), slice() e substr():

      • Permitem extrair partes de uma string.

    • replace() e replaceAll():

      • Substitui um trecho de texto por outro.

    • concat():

      • Permite conectar duas ou mais strings.

    • trim():

      • Remove os espaços em brancos no início e no final da string.

    • trimStart():

      • Remove espaços em branco do início da string.

    • trimEnd():

      • Remove espaços em branco do final da string.

    • padStart():

      • Preenche a string com caracteres à esquerda até atingir o comprimento desejado.

    • padEnd():

      • Preenche a string com caracteres à direita até atingir o comprimento desejado.

    • charAt():

      • Retorna o caractere em um índice específico da string.

    • charCodeAt():

      • Retorna o valor Unicode do caractere em um índice específico.

    • toFixed():

      • Formata um número com uma quantidade específica de casas decimais.

    • parseInt() e parseFloat():

      • Converte uma string em um número inteiro ou de um número

    • isNaN():

      • Verifica se um valor não é um número (NaN).

  • Number

    Number é usado para representar números inteiros e números decimais.

    • Declarando Variáveis Numéricas:

      • Usando as palavras-chaves var, let e const, você consegue declarar variáveis numéricas.

    • Operações Numéricas:

      • Soma: +
      Subtração: -
      Multiplicação: *
      Divisão: /

    • Conversão de Tipos:

      • JavaScript permite a conversão de tipos automaticamente, mas às vezes é necessário converter explicitamente um número para uma string ou vice-versa. Você pode fazer isso usando funções como toString() e Number().

  • Eventos

    Os eventos em JavaScript mostram ações em elementos que ocorrem na página web, como a passagem do mouse sobre uma imagem, cliques em botões ou pressionar uma tecla. São interações do usuário com o conteúdo da página que desencadeiam ações específicas.

    • onBlur()

      • Remove o foco do elemento.

    • onChange:

      • Muda o valor do elemento.

    • onClick:

      • O elemento é clicado pelo usuário.

    • onFocus:

      • O elemento é focado.

    • onKeyPress:

      • O usuário pressiona uma tecla sobre o elemento.

    • onLoad:

      • Carrega o elemento por completo.

    • onMouseOver:

      • Define ação quando o usuário passa o mouse sobre o elemento.

    • onMouseOut:

      • Define ação quando o usuário retira o mouse sobre o elemento.

    • onSubmit:

      • Define ação ao enviar um formulário.

  • Estruturas de repetição

    • for:

      • É um looping para executar um número específico de vezes. Consiste em três partes: inicialização, condição e incremento.

    • for in:

      • Usamos para repetir as propriedades de um objeto. Ela percorre todas as chaves do objeto.

    • for of:

      • Usamos para iterar sobre elementos iteráveis, como arrays e strings, fornecendo acesso direto aos valores dos elementos.

    • while:

      • Usamos para repetir um bloco de código enquanto a condição for verdadeira.

    • do while:

      • Ele verifica a condição só após a execução do bloco de código, garantindo que o código seja executado pelo menos uma vez, mesmo se a condição for falsa.

  • Estruturas de condição

    As estruturas condicionais são utilizadas para a tomada de decisão em um algoritmo. Quando uma expressão retorna verdadeiro ou falso, o algoritmo executa o bloco de comandos correspondente ao resultado. Por outro lado, as estruturas de repetição são úteis quando precisamos executar um bloco de comandos N vezes até que uma condição seja atendida.

    • if:

      • Significa (se) e permite executar um bloco de código se a condição dentro dos parênteses for verdadeira.

    • else:

      • Permite executar um bloco de código se a condição do if for falsa.

    • else if:

      • Permite executar um bloco de código se a condição do if for falsa e a condição do else if for verdareira. Pode usar else if o quanto quiser.

    • Switch Case:

      • A estrutura switch case em JavaScript é usada para criar um bloco de código que executa diferentes ações com base no valor de uma expressão. É uma forma eficiente de lidar com várias condições diferentes de forma mais legível do que uma série de instruções if...else.

      • Boolean indica se é verdadeiro (true) ou falso (false). Este tipo de dado é frequentemente usado em programação quando você precisa de um valor que pode ser apenas um de dois valores, como SIM/NÃO, LIGADO/DESLIGADO.

      • Date utilizado para definir datas, usando datas a partir de 1º de janeiro de 1970 (UTC). Para definir é utilizado new para indicar junto com o objeto date.

      • O construtor de Error cria um objeto de erro. Os objetos Error são lançadas quando erros de tempo na execução ocorre. O objeto Error também pode ser usado como objeto base para exceções definidas para o usuario. Estrutura:

      new Error([message[, fileName[, lineNumber]]])

      • Função é um bloco de código projetado para realizar uma tarefa específica. Uma função é executada quando “algo” a invoca (chama). As funções são fundamentais em Javascript, pois permitem que você escreva código uma vez e o reutilize em vários lugares. Exemplo:

      console.log("Olá, mundo!");
      }

      • JSONListKeys

        • Lista os nomes de objetos dos dados JSON para um elemento específico por um nome de objeto.

      • JSONListValues

        • Lista os valores em dados JSON para um elemento especificado por um nome de objeto.
