CSS (Cascading Style Sheets)

CSS é uma linguagem utilizada para estilizar elementos HTML em uma página web. Ele define a apresentação e o layout dos elementos. Com CSS, é possível controlar cores, fontes, espaçamentos, tamanhos e posicionamentos dos elementos.

Principais conceitos de CSS:

  • Seletores: identificam os elementos HTML a serem estilizados
  • Propriedades: definem o estilo dos elementos selecionados
  • Valores: são atribuídos às propriedades para modificar a aparência dos elementos
  • Cascata e herança: determinam como os estilos são aplicados e herdados pelos elementos

  1. Seletores e Propriedades:

    Seletores de Elementos, Classes, IDs, Propriedades de Estilo.

    Seletores de Elementos:

    Selecionam elementos com base no nome do elemento. Exemplo:
    p {
    color: red;
    }
    Aplica a cor vermelha a todos os parágrafos.

    Seletores de Classes:

    Selecionam elementos com base no atributo de classe. Exemplo:
    .myClass {
    color: red;
    }
    Aplica a cor vermelha a todos os elementos com a classe “myClass”.

    Seletores de IDs:

    Selecionam elementos com base no atributo de ID. Exemplo:
    #myID {
    color: red;
    }
    Aplica a cor vermelha ao elemento com o ID “myID”.

    Propriedades de Estilo:

    Definem o estilo dos elementos. Exemplo:
    color, font-size, background, etc .

  2. Modelo de Caixa:

    Margens, Bordas, Preenchimentos, Box Model.

    Margens (Margin):

    Espaço em torno de um elemento. Exemplo:
    margin: 10px;.

    Bordas (Border):

    Contorno em torno de um elemento. Exemplo:
    border: 1px solid black;.

    Preenchimentos (Padding):

    Espaço entre o conteúdo de um elemento e sua borda. Exemplo:
    padding: 10px;.

    Box Model:

    Descreve o design e layout. Inclui margin, border, padding, e o conteúdo real.

  3. Posicionamento e Layout:

    Display (block, inline, inline-block), Posicionamento (relative, absolute, fixed), Flexbox, Grid Layout.

    Display:

    Define como um elemento deve ser exibido. Exemplos:
    block, inline, inline-block.

    Posicionamento:

    Define como um elemento é posicionado. Exemplos:
    relative, absolute, fixed.

    Flexbox:

    Método de layout para alinhar itens em colunas e linhas. Exemplo:
    display: flex;.

    Grid Layout:

    Método de layout para alinhar itens em uma grade. Exemplo:
    display: grid;.

  4. Unidades e Valores:

    Unidades de Medida (px, em, rem, %, vw, vh), Valores Relativos e Absolutos.

    Unidades de Medida:

    Define o tamanho de vários aspectos. Exemplos:
    px, em, rem, %, vw, vh.

    Valores Relativos e Absolutos:

    em e rem são relativos ao tamanho da fonte, enquanto px e % são valores absolutos.

  5. Cascata e Herança:

    Prioridade de Estilos, Herança de Estilos.

    Prioridade de Estilos:

    Determinada pela especificidade do seletor e pela ordem dos estilos no CSS.

    Herança de Estilos:

    Alguns estilos são herdados pelos elementos filhos de um elemento pai.

  6. Estilização Avançada:

    Transformações, Transições, Animações, Gradientes, Sombras, Tipografia.

    Transformações:

    Permite girar, escalar, inclinar ou mover um elemento. Exemplo:
    transform: rotate(20deg);.

    Transições:

    Cria uma mudança gradual entre diferentes estados de um elemento. Exemplo:
    transition: all 2s;.

    Animações:

    Permite animar a transição entre estilos. Exemplo:
    animation: myAnimation 5s infinite;.

    Gradientes:

    Permite exibir transições suaves entre duas ou mais cores especificadas. Exemplo:
    background: linear-gradient(red, yellow);.

    Sombras:

    Aplica sombras aos elementos. Exemplo:
    box-shadow: 5px 5px 5px grey;.

    Tipografia:

    Estiliza o texto. Exemplo:
    font-size: 20px; font-family: Arial;.

  7. Responsividade e Media Queries:

    Design Responsivo, Media Queries.

    Design Responsivo:

    Faz com que o layout se adapte ao tamanho da tela.

    Media Queries:

    Aplica estilos diferentes para diferentes dispositivos ou tamanhos de tela. Exemplo:
    @media screen and (max-width: 600px) {...}.

  8. Frameworks e Pré-processadores:

    Bootstrap, Sass, Less.

    Bootstrap:

    Framework CSS para desenvolvimento responsivo e móvel.

    Sass:

    Pré-processador CSS que permite usar variáveis, aninhamento, mixins e muito mais.

    Less:

    Outro pré-processador CSS semelhante ao Sass.
